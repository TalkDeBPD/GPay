# GPAY
## 用到的库 best-pay-sdk
### 一款mc支付插件
#### 直接到达自己的支付宝和微信账号 
##### 前提是你有 商户号和当面付

# 配置
## key.yml ( 填入自己的微信商户号 appid 和支付宝等配置 即可收款)
![](https://www.xbaimiao.com/file/key.png)

> 微信申请前往微信商户平台<br>
> 支付宝请前往 商户中心

# 命令
> /gpay alipay 金额 <创建一个支付宝订单并将二维码以地图方式发送到玩家手上><br>
> /gpay wxpay 金额 <创建一个微信订单并将二维码以地图方式发送到玩家手上>
